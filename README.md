# Data-analysis-portfolio
A portfolio of data analysis projects using Python, pandas, and Power BI

## Overview
This project analyses divorce trends in South Africa using official data from Statistics South Africa and DataFirst (UCT).

## Objectives
- Identify trends in divorce cases over time
- Determine provinces with the highest divorce rates
- Analyse common grounds for divorce

## Tools Used
- Python
- pandas
- Power Bi
- Excel

## Data Cleaning
- Handled missing and unspecified categorical values using probability-based imputation
- Validated provinces and removed inconsistencies

## Key Findings
- Divorce cases increased steadily from 1980 to 2018
- Gauteng and Western Cape have the highest divorce rates
- Adultery and relationship breakdown are leading causes of divorce
